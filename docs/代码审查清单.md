# FIRE生活家 - 代码审查清单

> 版本：v1.0 | 最后更新：2026-05-12

## 一、UI/UX 检查项

### 1.1 文案一致性
- [ ] 页面入口菜单文案与页面标题完全一致
- [ ] 按钮文案符合用户习惯（确认/取消/保存/删除）
- [ ] 提示文案清晰明确
- [ ] 查阅并遵循《文案规范.md》

### 1.2 组件统一性
- [ ] 导航栏统一使用 `WdNavbar` 组件
- [ ] 按钮统一使用 `WdButton` 组件
- [ ] 弹窗统一使用 `WdDialog` 组件
- [ ] 输入框统一使用 `WdInput` 组件

### 1.3 交互体验
- [ ] 所有页面都有返回功能
- [ ] 加载状态有明确提示
- [ ] 空状态有友好提示
- [ ] 操作成功/失败有反馈

---

## 二、代码质量检查项

### 2.1 TypeScript 规范
- [ ] 类型定义完整，避免使用 `any`
- [ ] 导入语句有序且无重复
- [ ] 变量/函数命名清晰易懂

### 2.2 Vue 规范
- [ ] 组件命名符合规范
- [ ] `setup` 语法使用正确
- [ ] 响应式数据使用 `ref`/`computed` 合理

### 2.3 目录结构
- [ ] 新文件放在正确的目录下
- [ ] 遵循现有目录结构
- [ ] 不随意修改现有文件路径
