## 功能概述
资产总览页是用户了解个人财务状况的核心页面，集中展示净资产、各类资产占比、各账户余额等信息。顶部大字显示净资产（现金+投资+固定资产净权益+折旧资产-负债总计），下方展示月度趋势、资产配置和各账户卡片列表。

## 用户故事
作为用户，我希望在一个页面看到我的净资产、各类资产占比、以及每个账户的余额和价值，这样我可以快速了解自己的财务全貌，而不需要去计算每个账户的数字。

### 资产分类说明

| 分类 | 来源 | 余额/价值含义 |
|------|------|--------------|
| 存款 | 现金类账户余额总和 | 账户余额 |
| 投资 | 投资类账户市值总和 | 当前市值 |
| 房产 | 固定资产净权益总和 | 估值 - 关联负债 |
| 折旧资产 | 折旧资产当前价值总和 | 自动计算的当前价值 |
| 负债 | 所有负债账户余额总和 | 剩余本金 |

### 交互流程

1. **页面加载**
   - 显示 Loading 态
   - 请求账户列表和汇总数据
   - 渲染净资产、趋势图、配置图、账户列表

2. **净资产趋势图**
   - 点击趋势区域放大显示（可选）
   - 显示最近12个月的数据点
   - X轴：月份，Y轴：净资产金额

3. **资产配置环形图**
   - 点击分类区块显示详细数据
   - 分类：存款（现金类）、投资（投资类）、房产（固定资产净权益）、折旧资产、负债
   - 各分类颜色：
     - 存款：#00BFFF（卡布里蓝）
     - 投资：#19BE6B（绿色）
     - 房产：#0099CC（深蓝）
     - 折旧资产：#67C8FF（浅蓝）
     - 负债：#FA3534（红色）

4. **账户卡片列表**
   - 按类型分组展示
   - 资产类：现金类 → 投资类 → 固定资产 → 折旧资产
   - 负债类：信用卡/消费信贷 → 房贷/车贷 → 借款
   - 点击账户卡片 → 进入账户详情页
   - 支持折叠/展开分组

5. **FIRE 进度卡片**
   - 点击进入 FIRE 进度面板
   - 显示当前完成度和预计达成时间

6. **固定资产区块特殊展示**
   - 显示"估值待更新"提示（超过6个月）
   - 显示增值/贬值信息

7. **折旧资产区块特殊展示**
   - 显示当前总价值和本月折旧额
   - "查看全部"入口

### 状态变化

| 状态 | 触发条件 | 行为 |
|------|----------|------|
| Loading | 页面加载中 | 显示骨架屏/Loading |
| 空态 | 无账户数据 | 显示引导"添加账户开始记账" |
| 正常 | 有数据 | 显示完整资产总览 |
| 净资产为负 | 负债大于资产 | 净资产显示红色 |
| 趋势图加载中 | 请求趋势数据 | 显示小loading |
| 资产类为空 | 资产类账户为空 | 仅显示负债类 |
| 固定资产待更新 | 某固定资产超过6个月未更新 | 显示"估值待更新" |

## 数据结构

### 页面数据接口
```typescript
// GET /api/assets/overview
interface AssetsOverviewResponse {
  code: number;
  data: {
    summary: {
      netAssets: number;              // 净资产
      totalAssets: number;            // 资产总计
      totalLiabilities: number;       // 负债总计
      totalCash: number;              // 现金类总额
      totalInvestment: number;        // 投资类市值总额
      totalFixedNetEquity: number;    // 固定资产净权益总额
      totalDepreciating: number;      // 折旧资产当前价值总额
      monthlyDepreciation: number;    // 本月折旧总额
    };
    trend: {
      months: string[];               // ['2024-01', '2024-02', ...]
      values: number[];               // [100000, 105000, ...]
    };
    allocation: {
      category: string;               // 'cash' | 'investment' | 'fixed' | 'depreciating' | 'liability'
      label: string;                   // 显示名称
      amount: number;                 // 金额
      percentage: number;             // 占比
      color: string;                  // 颜色
    }[];
    accounts: {
      cash: Account[];                // 现金类账户
      investment: Account[];           // 投资类账户
      fixed: FixedAsset[];             // 固定资产
      depreciating: DepreciatingAsset[]; // 折旧资产
      liability: Account[];           // 负债类账户
    };
    fireProgress?: FireProgress;       // FIRE进度（可选）
  };
}

// 现金类/投资类/负债类账户
interface Account {
  id: string;
  name: string;
  icon: string;
  type: 'asset' | 'liability';
  subType: string;
  currentBalance: number;
  // 投资类额外
  costBasis?: number;
  marketValue?: number;
  profitLossRate?: number;
  // 负债类额外
  creditLimit?: number;
}

// 固定资产（简化）
interface FixedAsset {
  id: string;
  name: string;
  icon: string;
  category: string;
  netEquity: number;                  // 净权益
  currentValue: number;                // 当前估值
  appreciationRate: number;           // 增值率
  isValuationOverdue: boolean;         // 估值是否过期
}

// 折旧资产（简化）
interface DepreciatingAsset {
  id: string;
  name: string;
  icon: string;
  category: string;
  currentValue: number;               // 当前价值
  depreciationProgress: number;       // 折旧进度 0-100
  monthlyDepreciation: number;        // 月折旧额
  status: 'active' | 'disposed';
}

// FIRE进度数据（预留）
interface FireProgress {
  currentNetAssets: number;
  targetNetAssets: number;
  completionRate: number;            // 0-100
  estimatedDate: string;               // '2032-06-01'
}
```

## 边界情况

1. **净资产为负**
   - 净资产数字显示红色 #FA3534
   - 趋势图数据可能也是负数

2. **资产配置某类为0**
   - 环形图不显示该分类
   - 占比为0不显示

3. **固定资产超过6个月未更新**
   - 卡片底部显示"估值待更新"
   - 提示用户去更新估值

4. **折旧资产全部已处置**
   - 区块仍显示，但列表为空
   - 本月折旧额显示为0

5. **首次使用，无历史趋势**
   - 趋势图显示空状态
   - 提示"记录更多账单，生成趋势图"
