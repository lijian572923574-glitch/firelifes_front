## 功能概述
账户详情页展示单个账户的完整信息，包括当前余额、余额变动历史、以及对账功能。用户可以查看该账户的所有记账记录，并支持手动调整余额进行对账。

## 用户故事
作为用户，我希望查看某个账户的余额变动历史，了解钱从哪进、从哪出，并且可以在发现账实不符时手动调整余额进行对账，这样我可以精确管理每个账户。

### 交互流程

1. **查看账户详情**
   - 显示账户基本信息（图标、名称、余额）
   - 显示本月收支汇总
   - 加载余额变动记录（该账户的记账记录）

2. **余额变动列表**
   - 按日期分组（今天/昨天/MM月DD日）
   - 显示交易类型（支出/收入/调整）
   - 点击记录进入详情（可选）

3. **对账调整**
   - 点击"对账"按钮
   - 弹出对账弹窗
   - 输入调整金额（正负）和原因
   - 确认后生成一条"调整"类型的变动记录

4. **账户编辑**
   - 点击右上角"编辑"
   - 进入账户编辑页
   - 可修改名称/图标/显示设置

5. **删除账户**
   - 点击"..." → 删除
   - 二次确认弹窗
   - 显示关联记录数量

### 状态变化

| 状态 | 触发条件 | 行为 |
|------|----------|------|
| Loading | 页面加载中 | 显示骨架屏 |
| 空态 | 无变动记录 | 显示"暂无变动记录" |
| 正常 | 有数据 | 显示余额和变动列表 |
| 调整余额 | 点击对账确认 | 生成调整记录，更新余额 |
| 已删除 | 账户被删除 | 返回上一页 |

## 数据结构

### 账户详情接口
```typescript
// GET /api/accounts/:id - 获取账户详情
interface GetAccountDetailResponse {
  code: number;
  data: {
    account: Account;
    monthlySummary: {
      month: string;              // '2024-05'
      income: number;             // 本月收入
      expense: number;            // 本月支出
      adjustment: number;         // 本月调整
    };
    balanceChanges: BalanceChange[];
  };
}

interface BalanceChange {
  id: string;
  type: 'expense' | 'income' | 'adjustment' | 'transfer_out' | 'transfer_in';
  amount: number;                 // 正数表示增加，负数表示减少
  balanceAfter: number;          // 变动后余额
  category?: string;             // 关联分类名称
  note?: string;                 // 备注
  recordId?: string;             // 关联记账记录ID
  createdAt: string;             // 变动时间
}

// POST /api/accounts/:id/adjust - 余额调整
interface AdjustBalanceRequest {
  adjustment: number;            // 调整金额
  reason: string;                // 调整原因（必填）
}

// 对账调整记录
interface AdjustmentRecord {
  id: string;
  accountId: string;
  type: 'adjustment';
  amount: number;                // 正负调整金额
  balanceBefore: number;         // 调整前余额
  balanceAfter: number;          // 调整后余额
  reason: string;                // 调整原因
  createdAt: string;
}
```

## 边界情况

1. **账户不存在**
   - 显示404页面
   - 提示"账户不存在或已删除"

2. **无变动记录**
   - 显示空状态："还没有变动记录，开始记账吧"
   - 引导去记账页

3. **调整金额为0**
   - 禁止提交
   - 提示"调整金额不能为0"

4. **调整原因过长**
   - 最多100字符
   - 超出入提示

5. **并发修改**
   - 使用乐观更新
   - 冲突时提示刷新

6. **删除账户**
   - 删除后自动返回资产总览页
   - 提示"账户已删除"

7. **投资类账户（Phase 3）**
   - 显示持仓成本和当前市值
   - 显示浮动盈亏和收益率
