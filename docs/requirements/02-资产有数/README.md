# 02 - 资产有数

> 模块版本：v1.0 | 功能数：7 | 最后更新：2026-05-09

## 模块概述

资产有数的核心理念是"不用手动对账，记账自动更新资产，一目了然"。账户从"记账标签"升级为"一等公民"，成为个人财务管理的核心实体。用户可以创建、维护多种类型的账户，每种账户类型有不同的数据结构和管理方式。

## 模块数据流

```
记账记录
├── 选择账户 ───→ 更新账户余额
├── 选择分类
└── 提交成功

账户余额变化
├── 现金类：余额 = 收支
├── 投资类：市值波动手动更新
├── 固定资产：估值+负债管理
└── 折旧资产：自动计算折旧

净资产汇总
├── 资产总计 - 负债总计 = 净资产
└── 推送 FIRE 可期模块
```

## 账户类型说明

### 资产类
| 类型 | 说明 | 价值计算 |
|------|------|----------|
| 现金类 | 余额即价值 | 直接使用账户余额 |
| 投资类 | 市值波动 | 需要手动更新市值 |
| 固定资产 | 估值+负债 | 购买价 - 累计折旧 |
| 折旧资产 | 自动折旧 | 随时间自动减少 |

### 负债类
| 类型 | 说明 | 记录方式 |
|------|------|----------|
| 信用卡 | 消费后还款 | 负余额显示 |
| 贷款 | 分期/房贷等 | 显示剩余本金 |

## 与其他模块的关联

### 被依赖关系
