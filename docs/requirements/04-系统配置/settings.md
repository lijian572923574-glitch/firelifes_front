## 功能概述

设置页提供应用的个性化配置选项，包括基础设置（货币单位、记账提醒）、显示设置（FIRE时间换算、储蓄率显示）、数据管理（导出数据、清除缓存）和关于信息。

## 用户故事

作为用户，我希望能够根据个人偏好定制应用功能，如设置记账提醒时间、选择是否显示FIRE时间换算，同时能够导出我的数据备份或在需要时清除缓存。

### 交互流程

#### 1. 基础设置
- 点击货币单位 → 选择器选择 → 保存
- 点击记账提醒 → 开关切换 → 保存
- 点击起始页面 → 选择器选择 → 保存

#### 2. 显示设置
- 点击 FIRE 时间换算 → 开关切换 → 保存
- 点击储蓄率显示 → 开关切换 → 保存
- 点击 FIRE 进度卡片 → 开关切换 → 保存

#### 3. 数据导出
- 点击"导出为 CSV" → 生成文件 → 下载/分享
- 点击"导出为 Excel" → 生成文件 → 下载/分享

#### 4. 清除缓存
- 点击"清除本地缓存" → 确认弹窗 → 确认清除

#### 5. 清空数据
- 点击"清空所有数据" → 二次确认弹窗 → 输入"确认清空" → 执行清空

#### 6. 退出登录
- 点击"退出登录" → 确认弹窗 → 退出并跳转登录页

### 状态变化

| 状态 | 触发条件 | 行为 |
|------|----------|------|
| 设置保存 | 修改设置 | 保存到本地和服务端 |
| 开关切换 | 点击开关 | 实时更新状态 |
| 导出中 | 点击导出 | 显示 loading |
| 导出成功 | 生成文件成功 | 打开分享面板 |
| 导出失败 | 生成文件失败 | 显示错误提示 |
| 清除缓存 | 确认清除 | 显示成功提示 |
| 清空数据 | 确认清空 | 跳转登录页 |
| 未登录 | 设置页入口 | 跳转登录页 |

## 数据结构

### 用户设置
```typescript
interface UserSettings {
  // 基础设置
  currency: 'CNY' | 'USD' | 'EUR' | 'JPY';  // 默认 CNY
  reminderEnabled: boolean;                    // 记账提醒开关
  startPage: 'detail' | 'record' | 'assets' | 'fire';  // 起始页面

  // 显示设置
  fireTimeEnabled: boolean;                   // FIRE 时间换算
  savingsRateEnabled: boolean;                // 储蓄率显示
  fireProgressEnabled: boolean;                // FIRE 进度卡片

  // 同步设置
  autoSyncEnabled: boolean;                  // 自动同步
}

// API 接口
// GET /api/settings - 获取用户设置
// PUT /api/settings - 更新用户设置
```

## 边界情况

1. **未登录访问**
   - 设置页需要登录才能访问
   - 未登录时跳转到登录页

2. **网络错误保存设置**
   - 本地优先保存
   - 联网后自动同步到服务端

3. **导出失败**
   - 显示错误提示
   - 提供重试按钮

4. **清空数据后**
   - 清除本地所有数据
   - 跳转登录页
   - 显示"数据已清空"

5. **货币单位变更**
   - 所有金额显示自动转换
   - 不影响原始数据

6. **多设备设置同步**
   - 服务端保存最新设置
   - 登录时拉取最新设置
