## 功能概述
账户列表是账户体系的入口页面，展示所有账户，支持新增、编辑账户。新用户注册后自动创建3个默认账户。

## 用户故事
作为新用户，我希望注册后已有一些默认账户，这样我可以直接开始记账，不用手动创建。

## 一、色彩系统（极致简约风格）

### 1.1 主色（品牌色）
| Token | 色值 | 用途 |
|-------|------|------|
| `$uni-color-primary` | `#0D9488` | 主按钮、导航、账户图标、默认标签 |
| `#F8F9FA` | - | 页面背景（极浅灰，大量留白） |

### 1.2 功能色
| Token | 色值 | 用途 |
|-------|------|------|
| `$uni-color-success` | `#10B981` | 收入标识标签 |
| `$uni-color-error` | `#EF4444` | 删除按钮、负债余额 |

### 1.3 中性色（极致简约核心）
| Token | 色值 | 用途 |
|-------|------|------|
| `#FFFFFF` | - | 卡片背景（纯白） |
| `#94A3B8` | - | 分组标题文字（浅灰） |
| `#CBD5E1` | - | 描述文字、次要信息 |
| `$uni-text-color` | `#1E293B` | 主要文字、账户名称、余额 |

## 八、数据结构

### 账户数据结构
```typescript
type AccountType = 'cash' | 'investment' | 'fixed_asset' | 'depreciable_asset' | 'liability';

interface Account {
  id: string;
  userId: string;
  name: string;
  icon: string;
  type: AccountType;
  balance: number;
  description?: string;
  isDefaultExpense: boolean;
  isDefaultIncome: boolean;
  order: number;
  isVisible: boolean;
  isDeleted: boolean;
  createdAt: string;
  updatedAt: string;
}
```
