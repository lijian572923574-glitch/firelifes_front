## 功能概述
账户列表是账户体系的入口页面，展示所有账户，支持新增、编辑账户。

## 用户故事
作为用户，我希望在一个页面看到所有的财务账户，并能快速新增或修改账户信息。

### 账户卡片展示
每个账户卡片显示：
- **图标**：左侧显示账户图标
- **名称**：账户名称（加粗）
- **说明**：账户说明（可选，灰色小字）
- **余额**：右侧显示余额（资产绿色，负债红色）
- **箭头**：最右侧进入详情箭头

### 账户类型
| 类型标识 | 中文名称 | 说明 |
|---------|---------|------|
| asset | 资产类 | 现金、银行卡、投资等 |
| liability | 负债类 | 信用卡、贷款等 |

### 交互流程

1. **页面加载**
   - 显示所有账户列表
   - 无账户时显示空状态

2. **右上角"+"按钮**
   - 点击跳转到账户编辑页（新增模式）

3. **账户卡片点击**
   - 点击任意账户卡片，跳转到账户编辑页（编辑模式）

4. **长按/滑动删除（预留功能）**
   - 长按账户项进入可编辑状态
   - 滑动显示删除按钮

### 状态变化

| 状态 | 触发条件 | 行为 |
|------|----------|------|
| 空态 | 无账户 | 显示引导卡片"添加你的第一个账户" |
| 正常 | 有账户 | 显示账户列表 |
| 加载中 | 请求数据中 | 显示骨架屏 |

## 数据结构

### 账户数据结构
```typescript
interface Account {
  id: string;                      // UUID
  userId: string;                  // 用户ID
  name: string;                    // 账户名称
  icon: string;                    // 图标类型
  type: 'asset' | 'liability';     // 账户类型（资产/负债）
  balance: number;                 // 余额
  description?: string;            // 说明（可选）
  // 通用字段
  order: number;                   // 排序
  isVisible: boolean;              // 是否显示
  isDeleted: boolean;              // 软删除标记
  createdAt: string;
  updatedAt: string;
}
```
