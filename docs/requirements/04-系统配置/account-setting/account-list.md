## 功能概述
账户列表是账户体系的入口页面，展示所有账户，支持新增、编辑账户。新用户注册后自动创建3个默认账户。

## 用户故事
作为新用户，我希望注册后已有一些默认账户，这样我可以直接开始记账，不用手动创建。

### 6.2 列表更新

### 6.4 余额数字变化

| 动效 | 参数 |

| 进入时 | 数字从 0 滚动到目标值（`count-up`），`duration: 600ms ease-out` |

type AccountType = 'cash' | 'investment' | 'fixed_asset' | 'depreciable_asset' | 'liability';

interface Account {

  name: string;
  icon: string;
  type: AccountType;
  balance: number;
  description?: string;
  isDefaultExpense: boolean;
  isDefaultIncome: boolean;
  // 负债类账户专用字段
  originalPrincipal?: number;
  annualInterestRate?: number;
  repaymentMethod?: string;
  totalMonths?: number;
  remainingMonths?: number;
  repaymentDay?: number;
  linkedAssetAccountId?: string;
  // 通用字段
  order: number;
  isVisible: boolean;
  isDeleted: boolean;
  createdAt: string;
  updatedAt: string;
}

## 边界情况

1. **未登录状态** → 不显示账户列表，提示登录
2. **网络错误** → 显示错误状态，提供重试入口
3. **空状态** → 显示引导卡片"还没有添加账户"+ CTA 按钮
4. **删除默认账户** → 默认支出/收入账户仅显示编辑按钮，不显示删除按钮
5. **禁止横向滚动** → 页面根容器设置 `overflow-x: hidden`，避免与 WdSwipeAction 水平滑动冲突
6. **iPhone底部安全区** → 底部加 `safe-bottom` 区域 `height: calc(env(safe-area-inset-bottom) + 40rpx)`
