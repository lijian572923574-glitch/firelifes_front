## 功能概述
负债类账户不仅记录欠款余额，还支持配置贷款参数，实现自动计算每期还款额、自动拆分本金利息、自动记账。核心目标是让用户一次性录入贷款信息，之后全程自动处理。

## 用户故事
作为用户，我希望能为负债类账户配置贷款参数（利率、期数、还款方式等），让系统自动计算每月还款额并自动拆分本金和利息，这样我不用手动计算就能精准管理房贷等负债。

## 核心设计原则

```
┌─────────────────────────────────────────────────────────────┐
│                       设计原则总览                             │
├─────────────────────────────────────────────────────────────┤
│  1. 用户只维护静态基础信息，不维护每期还款金额                 │
│  2. 每期实际还款额、本息拆分，全靠公式自动运算                  │
│  3. 完美兼容：固定月供 / 递减月供 / 先息后本特殊还款           │
│  4. 房产固定资产账户保持不动，只动现金流 + 负债                 │
└─────────────────────────────────────────────────────────────┘
```

当账户类型选择「负债类」时，表单额外显示以下贷款参数字段：

| 贷款年利率 | ✅* | 数字 | 4.9 | *仅自动记账开启时必填，如4.9表示4.9% |
| 还款方式 | ✅* | 单选 | 等额本息 | *仅自动记账开启时必填：等额本息 / 等额本金 / 先息后本 / 灵活还款 |

| 每月还款日 | ⭕* | 日期选择 | - | *仅自动记账开启且非灵活还款时必填，1-28日 |

## 三种还款方式计算规则

### 基础公式定义

```
月利率 i = 年利率 ÷ 12 ÷ 100
P = 当期剩余本金
P总 = 原始贷款总本金
N = 总期数
n = 剩余期数
```

### 1. 等额本息（最常见房贷方式）

```
┌─────────────────────────────────────────────────┐
│  特点：每月还款额固定不变，前期利息多、后期本金多  │
├─────────────────────────────────────────────────┤
│  固定月供 M = P × [i(1+i)ⁿ] ÷ [(1+i)ⁿ - 1]      │
│  当月利息 = P × i                                │
│  当月还本 = M - 当月利息                         │
│  本月现金流出 = M（固定不变）                     │
└─────────────────────────────────────────────────┘
```

### 2. 等额本金

```
┌─────────────────────────────────────────────────┐
│  特点：月供逐月递减，每月还本金固定，利息逐月减少  │
├─────────────────────────────────────────────────┤
│  每月固定还本额 = P总 ÷ N                        │
│  当月利息 = 当期剩余本金 × i                     │
│  本月现金流出 = 固定本金 + 当月利息               │
│  （每期流出金额自动递减，无需人工设置）           │
└─────────────────────────────────────────────────┘
```

### 3. 先息后本

```
┌─────────────────────────────────────────────────┐
│  特点：前期只还利息，末期一次性还清本金           │
├─────────────────────────────────────────────────┤
│  正常还款期：本月流出 = 当期剩余本金 × i          │
│  最后一期结清：流出 = 剩余全部本金 + 当期利息     │
└─────────────────────────────────────────────────┘
```

### 4. 灵活还款（亲友借款模式）

```
┌─────────────────────────────────────────────────────────────┐
│  特点：无固定月供、无固定还款日、想还多少还多少、随时可还    │
│  适用场景：借亲戚朋友的钱、私人之间无固定期限借款             │
├─────────────────────────────────────────────────────────────┤
│  核心规则：                                                 │
│  1. 利率可选：无息 / 约定年利率                             │
│  2. 不自动触发还款日提醒，用户手动操作还款                   │
│  3. 每次还款金额用户自由输入（≥0）                           │
│  4. 系统自动拆分：有息时先扣息、后还本                       │
│  5. 无总期数、无剩余期数限制                                │
├─────────────────────────────────────────────────────────────┤
│  若约定有息：                                                │
│    应还利息 = 剩余本金 × 年利率 × 距上次还款天数 ÷ 365       │
│    本次还本金 = 本次还款总额 - 应还利息                      │
│    （利息滚入下一期，不产生复利）                            │
│                                                             │
│  若约定无息：                                                │
│    全部还款额直接冲减本金                                    │
│    利息 = 0                                                 │
└─────────────────────────────────────────────────────────────┘
```

## 自动记账流程

### 两种自动触发模式

| 还款方式 | 触发方式 |
|---------|---------|
| 等额本息 / 等额本金 / 先息后本 | 还款日定时自动触发 |
| 灵活还款 | 用户手动操作，系统自动计算拆分 |

### 模式1：还款日自动触发流程（前三种方式）

```
到每月还款日
    ↓
系统按贷款参数自动算出
    ↓
┌─────────────────────────────────┐
│  本期实际要扣多少钱              │
│  ├── 本期利息                   │
│  └── 本期归还本金               │
└─────────────────────────────────┘
    ↓
自动生成账务记录
    ↓
┌─────────────────────────────────┐
│  1. 流动资产减少 = 本期总流出   │
│  2. 负债账户减少 = 本期归还本金 │
│  3. 利息部分计入「房贷利息支出」│
└─────────────────────────────────┘
    ↓
自动更新剩余本金、剩余期数
    ↓
下月重新迭代计算
```

### 模式2：灵活还款手动操作流程

```
用户进入负债账户详情页
    ↓
点击 [还款] 按钮
    ↓
输入本次还款金额（自由输入）
    ↓
系统自动计算（按实际天数计息）
    ↓
┌─────────────────────────────────┐
│  本次还款：¥ 5,000              │
│  其中利息：¥ 200 （按实际天数）│
│  其中本金：¥ 4,800             │
│  还款后剩余本金：¥ 95,200      │
└─────────────────────────────────┘
    ↓
用户确认后生成记账记录
    ↓
┌─────────────────────────────────┐
│  1. 流动资产减少 = 还款总额     │
│  2. 负债账户减少 = 本金部分     │
│  3. 利息部分计入「利息支出」    │
└─────────────────────────────────┘
    ↓
更新 lastRepaymentDate 记录
```

### 完整账务分录（以房贷为例）

**场景**：月供 8000 元，其中本金 3000 元、利息 5000 元

| 账务项 | 金额变动 | 说明 |
|--------|---------|------|
| 流动资产（银行卡） | -8000 | 实际现金总流出 |
| 负债（住房贷款） | -3000 | 归还本金，负债减少 |
| 支出（房贷利息） | +5000 | 利息计入生活开销 |
| 固定资产（房产） | 0 | 房产账户全程不动 |

### 对报表的影响

| 报表类型 | 影响 | 数据表现 |
|---------|------|---------|
| 流水视角 | 总支出 8000 | 全部算"往外花钱" |
| 资产盘点视角 | 总资产不变 3000 + 减少 5000 | 本金还债调结构，利息纯消费 |
| 月度支出报表 | 计 5000 元支出 | 只有利息计入生活成本 |
| 资产负债表 | 资产 -8000、负债 -3000 | 净资产净减少 5000（利息部分） |

## 数据结构设计

### 后端 Entity 扩展

```typescript
// account.entity.ts 扩展字段
@Entity()
export class Account {
  // ... 原有字段
  
  // ===== 贷款自动记账相关字段 =====
  @Column({ default: false })
  autoRepaymentEnabled: boolean;          // 是否启用自动记账
  
  @Column({ type: 'decimal', precision: 15, scale: 2, nullable: true })
  originalPrincipal: number;              // 原始贷款总本金
  
  @Column({ type: 'decimal', precision: 5, scale: 2, nullable: true })
  annualInterestRate: number;             // 年利率，如 4.9 表示 4.9%
  
  @Column({ 
    type: 'enum', 
    enum: ['equal_principal_interest', 'equal_principal', 'interest_first', 'flexible'],
    nullable: true 
  })
  repaymentMethod: string;                // 还款方式
  
  @Column({ type: 'int', nullable: true })
  totalMonths: number;                    // 总还款期数
  
  @Column({ type: 'int', nullable: true })
  remainingMonths: number;                // 剩余还款期数
  
  @Column({ type: 'int', nullable: true })
  repaymentDay: number;                   // 每月还款日 1-28
  
  @Column({ nullable: true })
  linkedAssetAccountId: string;           // 关联的资产账户ID（如房产）
  
  @Column({ type: 'datetime', nullable: true })
  lastRepaymentDate: Date;                // 上次还款日期
}
```

### 前端类型定义

```typescript
// types/account.ts
export interface Account {
  // ... 原有字段

  autoRepaymentEnabled: boolean;

  repaymentMethod?: 'equal_principal_interest' | 'equal_principal' | 'interest_first' | 'flexible';

  remainingMonths?: number;

  lastRepaymentDate?: string;

// 还款计算结果

  principal: number;          // 本期归还本金

```

## API 接口设计

### 1. 计算当期还款额

```
GET /api/accounts/:id/calculate-repayment
```

**响应**：
```json
{
  "success": true,
  "data": {
    "totalPayment": 8000.00,
    "principal": 3000.00,
    "interest": 5000.00,
    "remainingPrincipal": 997000.00,
    "nextPaymentDate": "2026-06-21"
  }
}
```

### 2. 生成还款计划预览

```
GET /api/accounts/:id/repayment-schedule?months=12
```

**响应**：返回未来N个月的还款明细，用于预览验证

### 3. 执行自动还款记账

```
POST /api/accounts/:id/execute-repayment
```

（供定时任务调用，或用户手动触发立即还款）

## 定时任务设计

### 每日检查任务（仅针对前三种还款方式）

```
每日凌晨 01:00 执行
    ↓
遍历所有 autoRepaymentEnabled = true 
      AND repaymentMethod != 'flexible' 的负债账户
    ↓
检查今天是否是还款日（today.day === repaymentDay）
    ↓
如是 → 自动计算当期还款额并生成记账记录
    ↓
更新账户：balance（余额）、remainingMonths、lastRepaymentDate
    ↓
发送通知：本月房贷已自动记账，本金xxx，利息xxx
```

### 灵活还款说明

- 灵活还款模式不参与定时任务检查
- 不自动提醒还款日
- 用户主动进入账户详情页手动操作还款
- 还款金额完全由用户决定，系统只做利息计算和拆分

## 房产账户关联规则

### 关联目的

- 方便用户在查看房产资产时，一键看到对应房贷负债
- 支持一键计算：房产净值 = 房产市值 - 剩余房贷本金
- 还款记录可在关联资产账户下查看

### 关联表现

```
查看固定资产 - 自住房屋

│                                         │

│  └── 房产净值：350,000 元 （自动计算）   │

```

## UI 交互规范

### 1. 开关联动

- 默认「启用贷款自动记账」开关关闭
- 开启时才显示所有贷款参数字段
- 关闭时隐藏参数，清空已填值（需确认）

### 2. 还款方式字段联动

| 还款方式 | 显示字段 | 隐藏字段 |
|---------|---------|---------|
| 等额本息 / 等额本金 / 先息后本 | 原始本金、年利率、总期数、剩余期数、还款日 | - |
| 灵活还款 | 原始本金、年利率（可填0表示无息） | 总期数、剩余期数、还款日 |

### 3. 实时计算预览

**等额本息 / 等额本金 / 先息后本**：
```
┌─────────────────────────────────────┐
│  📊 还款预览（按当前参数计算）        │
│  首月月供：¥ 8,000.00                │
│  其中本金：¥ 3,000.00                │
│  其中利息：¥ 5,000.00                │
│  总利息支出：¥ 1,230,000.00          │
└─────────────────────────────────────┘
```

**灵活还款**：
```
┌─────────────────────────────────────┐
│  📊 灵活还款说明                      │
│  无固定还款日、无固定月供              │
│  随时可还，还多少自己定                │
│  年利率 0% = 无息借款                 │
│  有息时按实际天数计算利息              │
└─────────────────────────────────────┘
```

### 4. 期数联动校验

- 剩余还款期数 ≤ 总还款期数
- 还款日只能选 1-28 日（避免2月29/30/31问题）
- 灵活还款模式下期数字段不显示也不校验

## 验收标准

### 通用功能
- [ ] 负债类账户支持配置所有贷款参数
- [ ] 四种还款方式计算结果准确（等额本息/等额本金/先息后本/灵活还款）
- [ ] 利息正确计入支出报表，本金不计入消费支出
- [ ] 资产负债表总资产计算正确（本金还款不影响总资产）
- [ ] 关联资产账户后可一键查看房产净值

### 固定期限还款方式（前三种）
- [ ] 还款日自动触发记账，正确拆分为本金和利息
- [ ] 自动记账后负债余额和剩余期数正确更新
- [ ] 提供还款计划预览功能（未来N期）
- [ ] 定时任务正确检查并执行自动还款

### 灵活还款方式
- [ ] 期数、还款日等字段在灵活还款模式下隐藏
- [ ] 年利率可填0表示无息借款
- [ ] 用户可手动输入任意还款金额
- [ ] 有息时按实际天数（距上次还款日）计算利息
- [ ] 还款时正确先扣息后还本，剩余部分冲减本金
- [ ] 无息时全部还款额直接冲减本金
- [ ] 灵活还款不参与定时任务检查，不自动提醒
