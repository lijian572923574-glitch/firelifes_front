## 模块概述

系统配置是支撑整个应用的底层模块，包括用户身份认证、应用设置、分类管理、账户管理和数据同步功能。

## 模块数据流

```
认证模块
├── 微信一键登录 ──→ 获取 openid
├── 手机号登录 ───→ 验证码验证
└── Token 管理 ────→ 自动刷新/过期跳转

设置模块
├── 基础设置 ────→ 本地存储
├── 显示设置 ────→ 本地存储
└── 数据管理 ────→ 导出/清除

同步模块
├── 本地缓存 ───→ 离线记账
├── 云端同步 ───→ 联网后自动同步
└── 冲突处理 ───→ 以服务端为准
```

## 与其他模块的关联

### 依赖关系
- **无业务依赖**：系统配置是独立模块，不依赖业务模块

### 被依赖关系
- **记账省心**：依赖登录状态判断用户身份
- **资产有数**：依赖数据同步保证数据一致性
- **FIRE可期**：依赖设置项控制显示逻辑
