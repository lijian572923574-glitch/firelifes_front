# 分类设置 - 默认数据保护与UI升级

> 版本：v1.0 | 最后更新：2026-05-12 | 状态：进行中

---

## 一、需求背景

用户希望：
1. 默认数据不允许删除，只能禁用
2. 统一、协调、美观、精致的UI风格
3. 各个页面UI保持一致

---

## 二、功能需求

### 2.1 分类组管理

| 需求项 | 说明 |
|--------|------|
| 默认分类组保护 | 用户初始化时创建的默认分类组 `isUserCreated: false`，不允许删除 |
| 禁用/启用 | 所有分类组都可以禁用/启用，禁用后在记账时不显示 |
| 视觉区分 | 默认分类组有特殊标识，禁用状态有视觉反馈 |
| 用户自创分类组 | 用户自己创建的分类组可以完全删除/编辑/禁用 |

### 2.2 子分类管理

| 需求项 | 说明 |
|--------|------|
| 默认子分类保护 | `isUserCreated: false` 的子分类不允许删除 |
| 禁用/启用 | 所有子分类都可以禁用/启用 |
| 视觉区分 | 默认子分类有特殊标识，禁用状态有视觉反馈 |

---

## 三、视觉设计规范

严格遵循 [小米汽车蓝色设计规范](../../设计规范.md)：

### 3.1 色彩
- 主色：`#00BFFF`（小米汽车蓝）
- 背景色：`#F8F9FA`
- 卡片背景：`#FFFFFF`
- 主要文案：`#333333`
- 次级文案：`#666666`
- 辅助文案：`#999999`

### 3.2 通用卡片样式
```css
.card {
  background: #FFFFFF;
  border-radius: 16rpx;
  box-shadow: 0 4rpx 12rpx rgba(0, 0, 0, 0.04);
  padding: 28rpx;
  transition: all 150ms ease;
}
.card:active {
  transform: scale(0.99);
  background: rgba(0, 191, 255, 0.03);
}
```

### 3.3 禁用状态样式
- 透明度：60%
- 文字变灰
- 可以有禁用标识（如眼睛关闭图标）

### 3.4 默认标识样式
- 小徽章：`#00BFFF` 背景 + 白色文字 "默认"
- 圆角：`8rpx`
- 字号：`20rpx`

```
┌─────────────────────────┐
│  [←]  分类设置    [+]   │  ← Navbar
├─────────────────────────┤
│                         │
│  ┌───────────────────┐  │
│  │ [默认] 饮食消费 🔘│  ← 默认分类组卡片

│  │ 居家出行        🔘│  ← 用户自创卡片
│  │        编辑 · 禁用│  │
```

#### 交互说明
1. **点击卡片区域** → 进入子分类列表
2. **点击「编辑」** → 编辑分类组名称
3. **点击「禁用/启用」** → 切换状态
4. **点击「删除」** → 仅用户自创分类组显示此按钮

## 六、API 接口

### 6.1 分类组相关

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/category/groups` | GET | 获取用户分类组列表 |
| `/api/category/groups` | POST | 创建分类组 |
| `/api/category/groups/:id` | PUT | 更新分类组名称 |
| `/api/category/groups/:id/toggle` | POST | **新增**：切换启用/禁用状态 |
| `/api/category/groups/:id` | DELETE | 删除分类组（仅用户自创） |

### 6.2 子分类相关

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/category/group/:groupId/categories` | GET | 获取分类组下的子分类 |
| `/api/category/categories` | POST | 创建子分类 |
| `/api/category/categories/:id` | PUT | 更新子分类 |
| `/api/category/categories/:id/toggle` | POST | **新增**：切换启用/禁用状态 |
| `/api/category/categories/:id` | DELETE | 删除子分类（仅用户自创） |
