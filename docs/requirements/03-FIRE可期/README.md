# 03 - FIRE可期
> 模块版本：v1.1 | 功能数：6 | 最后更新：2026-05-15 | 中文名称：财务自由目标追踪与进度管理模块

## 模块概述

FIRE可期的核心理念是"基于真实数据算出你的FIRE日期，让目标可感知"。通过设定FIRE目标、追踪进度、计算储蓄率，让财务自由不再是遥不可及的梦想，而是每天都能看到进展的明确路径。

**核心策略**：不追求一次做完所有FIRE功能，而是采用"极简起步、逐步丰富"的策略。先在明细页嵌入储蓄率卡片和FIRE进度条，让用户每天看账单时都能感知FIRE距离。记账确认卡片中加入FIRE反馈，每次记账都是一次"距离自由还有多远"的提醒。

## 模块数据流

```
输入数据
├── 净资产（来自资产有数）
├── 月收入/月支出（来自记账省心）
└── 目标参数（FIRE目标金额）

FIRE 计算
├── 当前进度 = 净资产 / FIRE目标
├── 储蓄率 = (收入 - 支出) / 收入
└── 每次记账后即时刷新

输出展示
├── 明细页顶部FIRE进度条
├── 明细页储蓄率卡片
├── 记账确认卡片FIRE反馈
└── 后续：独立FIRE进度面板
```

## 核心公式

### FIRE 目标计算
```
FIRE 目标 = 目标月支出 × 12 ÷ 安全提现率(SWR)
          = ¥5,000 × 12 ÷ 4% = ¥1,500,000
默认：月支出取近3月均值，SWR默认4%
```

### 储蓄率计算
```
储蓄率 = (月收入 - 月支出) / 月收入 × 100%
```

### FIRE进度
```
当前进度 = 当前净资产 / FIRE目标 × 100%
```

## 与其他模块的关联

### 依赖关系
- **记账省心**：依赖支出数据计算月均支出、储蓄率
- **资产有数**：依赖净资产数据计算FIRE进度（唯一数据源）

### 被依赖关系
- **记账省心**：FIRE目标反馈消费决策，提醒用户控制支出
