# 03 - FIRE可期

> 模块版本：v1.0 | 功能数：5 | 最后更新：2026-05-09

## 模块概述

FIRE可期的核心理念是"基于真实数据算出你的FIRE日期，让目标可感知"。通过设定FIRE目标、追踪进度、计算储蓄率，让财务自由不再是遥不可及的梦想，而是每天都能看到进展的明确路径。

## 模块数据流

```
输入数据
├── 净资产（来自资产有数）
├── 月支出（来自记账省心）
└── 目标参数
    ├── 目标月支出
    ├── 安全提现率 (SWR)
    ├── 预期投资回报率
    └── 计划退休年龄

FIRE 计算
├── 目标净资产 = 月支出 × 12 / SWR
├── 当前进度 = 当前净资产 / 目标净资产
├── 储蓄率 = (收入 - 支出) / 收入
└── FIRE 日期 = 基于储蓄率和投资回报率计算

输出展示
├── 进度百分比
├── 预计 FIRE 日期
├── 月度 FIRE 报告
└── 消费建议（基于储蓄率）
```

## 核心公式

### FIRE 目标计算
```
FIRE 目标 = 目标月支出 × 12 ÷ 安全提现率
           = ¥5,000 × 12 ÷ 4% = ¥1,500,000
```

### 储蓄率计算
```
储蓄率 = (月收入 - 月支出) / 月收入 × 100%
```

### FIRE 时间估算
```
n = ln(1 + (FIRE目标 - 当前净资产) × r / 月储蓄) / ln(1 + r)
其中 r = 年投资回报率 / 12
```
