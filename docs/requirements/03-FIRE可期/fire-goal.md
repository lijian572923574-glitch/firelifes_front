## 功能概述
FIRE 目标设定允许用户定义自己的 FIRE（Financial Independence, Retire Early）财务自由目标，包括目标月支出、安全提现率、投资回报率等参数，系统自动计算需要积累的目标净资产。

## 用户故事
作为用户，我希望设定自己的 FIRE 目标，系统帮我算出我需要多少钱才能实现财务自由，这样我就有了一个明确的方向，知道每月需要存多少钱才能达成目标。

### 交互流程

1. **进入页面**
   - 有目标：显示当前目标数据
   - 无目标：显示空状态引导

2. **设置目标月支出**
   - 可手动输入
   - 可点击"自动计算"从历史记账数据中取均值
   - 可选择近3个月/6个月/12个月
   - 自动排除异常大额支出（可选）

3. **设置安全提现率**
   - 滑动选择，范围 1%-6%
   - 常用预设：3%（保守）、4%（标准）、5%（激进）
   - 默认值：4%
   - 说明：安全提现率指退休后每年从本金中提取的比例

4. **设置预期投资回报率**
   - 滑动选择，范围 1%-12%
   - 默认值：7%
   - 说明：基于历史股市平均回报

5. **设置退休年龄**
   - 滑动选择，范围 30-70岁
   - 默认值：55岁
   - 可输入具体数字

6. **实时计算目标净资产**
   - 公式：目标净资产 = 目标月支出 × 12 ÷ 安全提现率
   - 例如：¥5,000 × 12 ÷ 4% = ¥1,500,000

7. **保存目标**
   - 验证必填项
   - 保存到服务端
   - 返回上一页或刷新 FIRE 进度面板

### 状态变化

## 数据结构

### FIRE 目标数据结构
```typescript
interface FireGoal {
  id: string;
  userId: string;
  targetMonthlyExpense: number;  // 目标月支出
  safeWithdrawalRate: number;    // 安全提现率 (0.01-0.06)
  expectedReturnRate: number;    // 预期投资回报率 (0.01-0.12)
  retirementAge: number;         // 退休年龄 (30-70)
  targetNetAssets: number;       // 目标净资产（计算得出）
  currentAge?: number;           // 当前年龄（可选）
  createdAt: string;
  updatedAt: string;
}

// GET /api/fire/goal - 获取 FIRE 目标
interface GetFireGoalResponse {
  code: number;
  data: {
    goal: FireGoal | null;       // null 表示未设定
  };
}

// POST /api/fire/goal - 创建/更新 FIRE 目标
interface SaveFireGoalRequest {
  targetMonthlyExpense: number;
  safeWithdrawalRate: number;
  expectedReturnRate: number;
  retirementAge: number;
}

// 计算月均支出（服务端）
interface CalculateAvgExpenseRequest {
  months: 3 | 6 | 12;
  excludeOutliers?: boolean;     // 是否排除异常值
}

interface CalculateAvgExpenseResponse {
  code: number;
  data: {
    avgMonthlyExpense: number;
    sampleMonths: number;
    dataPoints: {
      month: string;
      amount: number;
    }[];
  };
}
```

## 边界情况

1. **首次使用**
   - 显示空状态引导
   - 提供"一键设置"选项

2. **月均支出为0**
   - 提示"记账数据不足，请先记账"
   - 允许手动输入

3. **目标净资产计算溢出**
   - 月支出过大导致目标资产超过 10 亿
   - 提示"月支出过高，请确认"

4. **安全提现率过低/过高**
   - 1% 或 6% 时显示警告
   - 提示相应的风险

5. **退休年龄小于当前年龄**
   - 提示"退休年龄应大于当前年龄"
   - 或直接跳转到 FIRE 已达成状态

6. **未记账**
   - 无法使用自动计算
   - 仅允许手动输入

7. **目标已达成**
   - 提示"恭喜！您已达成 FIRE 目标"
   - 引导进入"FIRE 已达成"页面
