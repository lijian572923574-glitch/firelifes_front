## 功能概述
储蓄率追踪功能帮助用户了解每月收入中真正存下来的比例。储蓄率 = (月收入 - 月支出) / 月收入，以可视化图表展示历史趋势，并用颜色分级提示储蓄健康状况。

## 用户故事
作为用户，我希望看到自己每月存了多少钱、储蓄率是多少，以及储蓄率的变化趋势，这样我可以评估自己的财务状况是否健康，有没有乱花钱。

### 交互流程

1. **查看本月储蓄率**
   - 明细页顶部显示本月储蓄率标签
   - 点击进入储蓄率趋势页

2. **查看年度累计**
   - 显示年度累计已存金额
   - 显示年度累计储蓄率

3. **查看趋势图**
   - 显示近12个月柱状图
   - 柱子高度 = 储蓄率
   - 柱子颜色按分级着色

4. **查看月度明细**
   - 列表展示每月数据
   - 显示收入、支出、储蓄率

5. **切换视图**
   - 切换月份/年份
   - 导出报告（可选）

### 储蓄率颜色分级

| 储蓄率 | 颜色 | 含义 | 建议 |
|--------|------|------|------|
| ≥ 50% | #19BE6B（绿色） | 非常健康 | 继续保持 |
| 30%-50% | #00BFFF（卡布里蓝） | 正常 | 稳步积累 |
| 10%-30% | #FF9800（橙色） | 偏低 | 注意开支 |
| < 10% | #FA3534（红色） | 危险 | 控制支出 |

### 状态变化

| 状态 | 触发条件 | 行为 |
|------|----------|------|
| Loading | 加载中 | 显示骨架屏 |
| 正常 | 有数据 | 显示储蓄率 |
| 储蓄率负数 | 支出 > 收入 | 显示红色，负数 |
| 储蓄率为0 | 收支平衡 | 显示灰色 |
| 无数据 | 当月无记账 | 显示"--" |
| 数据不足 | 不足12个月 | 显示已有数据 |

## 数据结构

### 储蓄率数据接口
```typescript
// GET /api/analytics/savings-rate
// GET /api/analytics/savings-rate?year=2024
interface GetSavingsRateResponse {
  code: number;
  data: {
    currentMonth: {
      month: string;               // '2024-05'
      income: number;
      expense: number;
      savings: number;            // income - expense
      rate: number;               // 0-1
      status: 'excellent' | 'good' | 'warning' | 'danger';
    };
    yearSummary: {
      totalIncome: number;
      totalExpense: number;
      totalSavings: number;
      avgRate: number;
    };
    monthlyData: {
      month: string;
      income: number;
      expense: number;
      savings: number;
      rate: number;
      status: string;
    }[];
  };
}

// 月度储蓄率计算逻辑
interface MonthlySavingsRate {
  month: string;                  // 'YYYY-MM'
  income: number;                 // 当月收入合计
  expense: number;               // 当月支出合计
  savings: number;               // 储蓄金额 = income - expense
  rate: number;                   // 储蓄率 = savings / income
  // rate 为负数时表示入不敷出
  // rate = 0 时表示收支平衡
}
```

## 边界情况

1. **当月无收入**
   - 储蓄率显示 "--"
   - 提示"本月暂无收入"

2. **当月无支出**
   - 储蓄率 = 100%
   - 显示绿色

3. **当月无任何记录**
   - 储蓄率显示 "--"
   - 提示"本月暂无记账数据"

4. **储蓄率为负数**
   - 显示负数百分比
   - 颜色使用红色
   - 提示"本月入不敷出"

5. **储蓄率为0**
   - 收支刚好平衡
   - 颜色使用灰色

6. **数据不足12个月**
   - 显示已有数据
   - 趋势图可滚动查看

7. **年度切换**
   - 切换到无数据年份
   - 显示空状态

8. **跨年数据**
   - 年度累计只计算当年数据
   - 趋势图可以跨年显示

9. **收入极少导致储蓄率异常**
   - 收入 100，支出 50，储蓄率 50%
   - 收入 1000，支出 990，储蓄率 1%
   - 都按实际计算，不做额外处理
