## 功能概述
月度 FIRE 报告自动生成上月财务总结，包括收支情况、净资产变化、FIRE 进度推进、距离目标变化等，并以精美的卡片式页面呈现，支持分享。

## 用户故事
作为用户，我希望每月收到一份自己的财务报告，总结这个月的表现、FIRE 进度变化，让我感受到自己的进步，同时发现可以改进的地方。

### 交互流程

1. **报告生成**
   - 每月1日凌晨自动生成上月报告
   - 用户也可手动触发生成

2. **查看报告**
   - 进入报告页
   - 查看各项数据卡片
   - 左右滑动切换月份

3. **分享报告**
   - 点击分享按钮
   - 生成图片分享
   - 可选平台：微信、朋友圈、图片保存

4. **查看详细**
   - 点击"查看详细 FIRE 进度"
   - 进入 FIRE 进度面板

### 状态变化

| 状态 | 触发条件 | 行为 |
|------|----------|------|
| Loading | 生成中 | 显示加载动画 |
| 正常 | 有数据 | 显示完整报告 |
| 无上月数据 | 首次使用 | 显示引导"开始记账" |
| 已分享 | 分享成功 | 提示"分享成功" |
| 加载失败 | 请求失败 | 显示重试按钮 |

## 数据结构

### 月度报告接口
```typescript
// GET /api/fire/reports/:month
// month 格式: 'YYYY-MM'
interface GetMonthlyReportResponse {
  code: number;
  data: {
    report: FireMonthlyReport;
  };
}

// POST /api/fire/reports/generate
// 手动生成报告
interface GenerateReportRequest {
  month: string;                  // 'YYYY-MM'
}

interface FireMonthlyReport {
  id: string;
  month: string;                  // '2024-04'
  generatedAt: string;           // 生成时间

  summary: {
    income: number;               // 本月收入
    expense: number;             // 本月支出
    savings: number;             // 本月储蓄
    savingsRate: number;          // 储蓄率
    savingsRateChange: number;    // 储蓄率变化
  };

  netAssets: {
    previousMonth: number;       // 上月净资产
    currentMonth: number;        // 本月净资产
    change: number;              // 变化金额
    changeRate: number;          // 变化率
  };

  fireProgress: {
    completionRate: number;      // 完成度
    progressChange: number;      // 进度推进
    remainingAmount: number;      // 距目标差额
    estimatedDate: string;       // 预计达成日期
  };

  topCategories: {
    categoryId: string;
    categoryName: string;
    icon: string;
    amount: number;
    percentage: number;
  }[];

  comparison: {
    expenseChange: number;       // 支出变化
    savingsRateChange: number;    // 储蓄率变化
    progressChange: number;       // 进度变化
  };

  isShared: boolean;             // 是否已分享
}

// 获取报告列表
interface GetReportListResponse {
  code: number;
  data: {
    reports: {
      month: string;
      summary: string;           // 摘要描述
    }[];
  };
}
```

## 边界情况

1. **首次使用**
   - 首月无对比数据
   - 显示"这是你的第一个月 FIRE 报告！"

2. **无上月数据**
   - 无法计算环比变化
   - 显示"--"

3. **报告未生成**
   - 页面显示"报告生成中..."
   - 提示稍后再试

4. **分享失败**
   - 显示错误提示
   - 提供重试按钮

5. **净资产减少**
   - 变化显示负数
   - 使用红色

6. **FIRE 进度后退**
   - 可能因为大幅消费
   - 显示后退百分比
   - 提示鼓励文案

7. **当月无收入**
   - 收入显示 ¥0
   - 储蓄率为负数

8. **报告过期**
   - 超过12个月的报告
   - 可归档或删除
