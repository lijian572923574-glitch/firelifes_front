## 功能概述
FIRE 进度面板以直观的圆环进度图展示用户当前净资产距离 FIRE 目标的完成度，同时显示预计达成日期、储蓄率趋势、三项关键指标等数据，帮助用户追踪 FIRE 进程。

## 用户故事
作为用户，我希望看到自己距离 FIRE 目标还有多远，每个月看到进度向前推进，这会给我坚持下去的动力，让我知道只要坚持记账和储蓄，梦想一定会实现。

### 交互流程

1. **查看进度概览**
   - 加载时显示圆环进度动画
   - 中心显示完成百分比
   - 下方显示当前净资产

2. **查看关键指标**
   - 三个卡片展示：净资产、月均支出、月均收入
   - 点击卡片可查看详情

3. **查看储蓄率**
   - 显示本月储蓄率
   - 显示年度平均储蓄率
   - 显示趋势箭头（上升/下降/持平）

4. **查看距目标差额**
   - 大字显示还需多少
   - 鼓励文案

5. **时间线交互**
   - 可左右滑动查看
   - 点击进入详细分析（可选）

6. **调整目标**
   - 点击进入 FIRE 目标设定页

### 状态变化

| 状态 | 触发条件 | 行为 |
|------|----------|------|
| Loading | 页面加载中 | 显示骨架屏 |
| 未设定目标 | FireGoal 为空 | 显示引导"设定 FIRE 目标" |
| 进行中 | 净资产 < 目标 | 显示进度圆环 |
| 已达成 | 净资产 >= 目标 | 显示达成庆祝页 |
| 超额 | 净资产 >> 目标 | 显示超额百分比 |

## 数据结构

### FIRE 进度数据
```typescript
// GET /api/fire/progress
interface GetFireProgressResponse {
  code: number;
  data: {
    summary: {
      currentNetAssets: number;      // 当前净资产
      targetNetAssets: number;        // 目标净资产
      completionRate: number;         // 完成度 0-100
      remainingAmount: number;        // 距目标差额
      estimatedDate: string;         // 预计达成日期 'YYYY-MM'
      status: 'in_progress' | 'achieved' | 'exceeded';
    };
    metrics: {
      currentNetAssets: number;
      avgMonthlyExpense: number;
      avgMonthlyIncome: number;
    };
    savingsRate: {
      currentMonth: number;           // 本月储蓄率
      yearAverage: number;           // 年度平均储蓄率
      trend: 'up' | 'down' | 'stable';
    };
    timeline: {
      startDate: string;              // 开始日期
      targetDate: string;             // 目标日期
      currentDate: string;            // 当前日期
      progressPercent: number;        // 时间进度
    };
  };
}

// 计算预计达成日期
interface EstimatedDate {
  date: string;                      // '2032-06'
  yearsRemaining: number;            // 剩余年数
  monthsRemaining: number;           // 剩余月数
  basedOn: {
    currentSavingsRate: number;
    expectedReturnRate: number;
  };
}
```

## 边界情况

1. **净资产为负**
   - 显示 0% 进度
   - 提示"先还清负债再开始 FIRE"

2. **未设定目标**
   - 显示空状态引导
   - 提供"立即设定"按钮

3. **目标已达成**
   - 显示庆祝页面
   - 显示"已超出目标 X%"
   - 可选择"调整更高目标"

4. **储蓄率为负**
   - 本月入不敷出
   - 显示红色警告
   - 提示"本月支出超过收入"

5. **预计日期超出退休年龄**
   - 提示"按当前进度，可能无法在目标年龄达成"
   - 建议调整目标或增加储蓄率

6. **数据不足**
   - 记账不满一个月
   - 提示"继续记账以获取更准确的预测"

7. **预计日期超过 100 年**
   - 按当前储蓄率永远无法达成
   - 提示"建议提高储蓄率或降低目标"

8. **净资产刚好等于目标**
   - 显示 100% 进度
   - 提示"恭喜达成 FIRE！"
