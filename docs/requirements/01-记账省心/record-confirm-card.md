## 功能概述
记账成功后显示一个确认卡片，包含分类图标、金额和成功提示。卡片2秒后自动消失，支持滑动关闭。这个组件是"继续记下一笔"功能的基础组成部分。

## 用户故事
作为用户，我希望在提交记账后得到清晰的视觉反馈，确认我的记录已被成功保存。确认卡片让我知道操作完成，无需担心是否遗漏。

### 交互流程
1. 记账提交成功，弹出确认卡片
2. 显示分类图标（大尺寸）、金额、分类名称
3. 底部进度条开始倒计时（2秒）
4. 用户可选择：
   - 点击"继续记一笔"：关闭卡片，保持记账页状态
   - 点击"去明细页"：立即跳转明细页
   - 滑动关闭：关闭卡片
   - 等待2秒：自动消失（不跳转）

### 状态变化
| 状态 | 触发条件 | 行为 |
|------|----------|------|
| 隐藏 | 页面初始 | 卡片不可见 |
| 显示 | 提交成功 | 卡片从底部滑入 |
| 倒计时 | 卡片显示 | 进度条从100%→0% |
| 关闭 | 点击按钮/滑动/超时 | 卡片消失 |

## UI 设计规范

### 布局
- 卡片：居中显示，宽度 560rpx，高度自适应
- 分类图标：96rpx × 96rpx
- 金额：48rpx，加粗
- 进度条：宽度 100%，高度 4rpx
- 按钮：高度 72rpx，圆角 36rpx

### 颜色
- 遮罩：rgba(0, 0, 0, 0.3)
- 卡片背景：#FFFFFF
- 卡片阴影：0 8rpx 32rpx rgba(0, 191, 255, 0.2)
- 金额文字：#333333（支出）/ #19BE6B（收入）
- 进度条：#00BFFF
- 主按钮背景：linear-gradient(135deg, #00BFFF, #0099CC)
- 次按钮文字：#999999

### 字体
- 分类名称：28rpx，#999999
- 金额：48rpx，#333333，加粗
- 成功提示：24rpx，#999999
- 按钮文字：28rpx，#FFFFFF

### 动效
- 卡片弹出：从底部滑入 + 缩放 0.9→1，时长 300ms
- 卡片关闭：淡出 + 向上滑出，时长 200ms
- 进度条：线性从右到左，2秒

## 边界情况
1. **快速切换分类**：卡片显示时如果金额变化，不更新卡片
2. **页面跳转时卡片显示中**：自动关闭
3. **App 被切后台**：暂停倒计时
4. **网络延迟**：提交按钮 loading 状态结束后再显示卡片
