# 01 - 记账省心
> 模块版本：v1.3 | 功能数：17 | 最后更新：2026-05-16 | 中文名称：快速记账与智能辅助功能模块

## 模块概述

记账省心是 FIRE生活家的核心基础模块，核心理念是"让记账零摩擦，3秒完成一笔"。通过极简的交互设计、智能的默认行为和完善的快捷功能，降低用户的记账心理负担，养成日常记账习惯。

**在三支柱中的地位**：记账省心是数据源头。用户每天记账产生的数据，自动流转到「资产有数」更新账户余额和折旧资产，最终汇聚到「FIRE可期」计算储蓄率和FIRE进度。没有记账数据，整个系统无从运转。

## 三支柱全链路审视

### 记账省心的核心价值——数据源头

```
用户每天记账 ──→ 数据积累 ──→ 资产沉淀 ──→ 净资产增长 ──→ 抵达FIRE目标
  (记账省心)      (自动)      (资产有数)      (净资产引擎)     (FIRE可期)
```

### 已实现优化

| 方向 | 说明 | 状态 |
|------|------|------|
| 记账即资产 | 支出购买大件时一键记入折旧资产 | ✅已完成（AssetFields开关+事务双写） |
| 记账即反馈 | 每次记账后展示净资产+FIRE距离 | 🔵开发中（RecordConfirmCard） |
| 记账即省心 | 品类智能匹配、金额阈值提醒 | 🟡设计中 |
| 连续记账 | 记账成功后选择"继续记" | 🔵开发中（合并到确认卡片） |

## 模块数据流

```
用户输入
├── 时间选择
├── 账户选择 ───→ 关联资产模块
├── 金额输入
├── 分类选择
├── 记入资产开关 ───→ 展开折旧资产编辑区
├── 备注输入 ───→ 智能备注建议
└── 提交
    │
    ▼
记账成功确认卡片
├── 当前净资产 ───→ 来自资产有数
├── FIRE进度 ────→ 来自FIRE可期
├── 继续记一笔 ───→ 清空金额+备注，保留分类
└── 返回明细页 ───→ 跳转明细页

存储记录
├── 更新账户余额
├── 创建折旧资产（如开启开关）
├── 计入月度统计
└── 计算储蓄率（明细页展示）
```

## 与其他模块的关联

### 依赖关系
- **无外部依赖**：记账省心是基础层，不依赖其他模块

### 被依赖关系
- **资产有数**：记账数据自动更新对应账户余额、创建折旧资产
- **FIRE可期**：支出/收入数据用于储蓄率计算和FIRE进度追踪
