## 模块概述

记账省心是 FIRE生活家的核心基础模块，核心理念是"让记账零摩擦，3秒完成一笔"。通过极简的交互设计、智能的默认行为和完善的快捷功能，降低用户的记账心理负担，养成日常记账习惯。

## 模块数据流

```
用户输入
├── 时间选择
├── 账户选择 ───→ 关联资产模块
├── 金额输入
├── 分类选择
├── 备注输入 ───→ 智能备注建议
└── 提交
    │
    ▼
记账成功确认卡片
├── 继续记一笔 ───→ 清空金额+备注，保留分类
└── 去明细页 ────→ 跳转明细页

存储记录
├── 更新账户余额
├── 计入月度统计
└── 推送资产模块
```

## 与其他模块的关联

### 依赖关系
- **无外部依赖**：记账省心是基础层，不依赖其他模块

### 被依赖关系
- **资产有数**：记账数据自动更新对应账户余额
- **FIRE可期**：支出/收入数据用于储蓄率计算和FIRE进度追踪
