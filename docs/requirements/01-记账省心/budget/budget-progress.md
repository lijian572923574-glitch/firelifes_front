## 一、功能概述

预算进度页是用户查看预算执行情况的核心入口。展示当月常规预算的消耗进度、专项准备金的累积进度，以及超支预警信息。帮助用户实时掌握消费节奏，及时调整支出行为。

### 1.1 核心价值

- **实时反馈**：每笔记账后，budget.spent 实时更新，进度即时刷新
- **视觉预警**：进度条颜色变化（绿→蓝→橙→红）直观反映预算健康度，v1.0 由前端判断
- **准备金追踪**（v1.5 新增）：专项准备金的累积进度一目了然
- **三支柱联动**：预算执行率 → 储蓄率 → FIRE进度

### 1.2 用户故事

作为用户，我希望打开预算页就能看到：
- 这个月总共花了多少，还剩多少额度
- 每个分类的预算用了多少，哪个快超支了
- 我的准备金存了多少，还差多少
- 每天平均还能花多少钱

### 2.2 进度条颜色规则

| spent / amount | 颜色 | 色值 | 含义 |
|----------------|------|------|------|
| < 60% | 🟢 绿色 | #00b894 | 预算健康，放心消费 |
| 60% ~ 80% | 🔵 蓝色 | #00BFFF | 正常消耗中 |
| 80% ~ 95% | 🟡 橙色 | #fdcb6e | ⚠️ 接近预算，注意控制 |
| 95% ~ 100% | 🟠 深橙 | #e17055 | 即将超支，建议节制 |
| ≥ 100% | 🔴 红色 | #d63031 | 已超支！ |

### 2.3 顶部汇总卡片数据口径

| 指标 | 计算方式 |
|------|----------|
| 总预算 | SUM(当月所有生效常规预算金额) |
| 已支出 | SUM(当月所有常规预算 spent) |
| 剩余 | 总预算 - 已支出 |
| 日均可用 | 剩余 ÷ 当月剩余天数 |
| 预计月末余额 | 总预算 - (已支出 ÷ 已过天数 × 当月总天数) |

## 三、交互设计

### 3.1 交互流程

```
1. 进入预算进度页
   ├── 加载当月预算数据（GET /api/budgets/overview/current）
   └── 渲染进度条动画（进度条从0过渡到实际值，0.6s缓动）

2. 下拉刷新
   └── 重新加载预算数据 → spent已更新 → 进度条动画重新过渡

3. 点击分类预算卡片
   ├── 展开使用明细：
   │   ├── 最近10笔该分类的支出记录（通过 GET /record?typeId=xx&month=xx 实时聚合）
   │   ├── 每笔扣减了多少预算
   │   └── "查看全部"跳转明细页（筛选该分类）
   └── 再次点击收起

4. 点击预警卡片
   └── 跳转到对应预算分类明细

5. 点击右上角⚙️图标
   └── 进入预算设置页（编辑预算金额、添加分类、管理准备金）

6. 点击准备金折叠区
   ├── 展开/收起准备金详情
   └── "管理准备金"跳转预算设置页（v1.5：跳转独立准备金管理页）

7. 预警区超出2条时
   ├── 最多显示2条 + "+N条更多预警 → [展开查看全部]"
   └── 点击展开显示全部预警列表
```

## 四、与记账模块的实时联动

### 4.1 记账后的预算进度刷新

```
用户在记账页完成一笔支出 →
  1. 后端事务内：写入 record + UPDATE budget.spent
  2. 后端返回 budgetImpact（受影响预算、usedPercentage）
  3. 明细页 FunctionBar 下方的 fire-mini-bar 旁增加轻量预算状态：
     "本月预算已用 75%，剩余 ¥2,500"
  4. 用户进入预算进度页 → 重新加载 → 进度条动画更新
```

```
明细页（index.vue）在 fire-mini-bar 和 FunctionBar 之间：
```

### 预警计算逻辑（v1.0 前端实现）

```typescript
// BudgetBar.vue 或 budget store 中
function getBudgetAlertLevel(spent: number, amount: number, threshold: number) {
  const pct = (spent / amount) * 100
  if (pct >= 100) return { level: 'danger', color: '#d63031', label: '已超支' }
  if (pct >= threshold) return { level: 'warning', color: '#fdcb6e', label: '接近预算' }
  if (pct >= 60) return { level: 'normal', color: '#00BFFF', label: '正常' }
  return { level: 'healthy', color: '#00b894', label: '健康' }
}
```

> v2.0 改为后端持久化预警（budget_alerts 表），支持推送通知。
