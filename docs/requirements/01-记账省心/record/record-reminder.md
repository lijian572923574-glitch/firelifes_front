## 功能概述
支持用户设置每日固定时间提醒记账（如每天21:00），支持多时间段设置。使用系统本地通知（uni-app push）推送提醒消息，设置入口在"我的"页面。

## 用户故事
作为用户，我经常忘记记账，尤其是一些小额支出。希望能设置每日提醒，到点收到推送通知提醒我记账，这样就不会遗漏任何一笔收支了。

### 交互流程
1. 用户进入"我的"页面，点击"记账提醒"
2. 进入提醒设置页面
3. 开启提醒开关
4. 添加提醒时间（可添加多个）
5. 选择提醒日期（每天/工作日/自定义）
6. 保存设置
7. 到点推送本地通知
8. 点击通知可直接进入记账页

### 状态变化
| 状态 | 触发条件 | 行为 |
|------|----------|------|
| 关闭 | 开关关闭 | 不推送通知 |
| 开启 | 开关打开 | 订阅推送 |
| 添加时间 | 点击添加 | 显示时间选择器 |
| 删除时间 | 点击删除 | 确认后移除 |
| 保存 | 点击保存 | 同步到系统通知 |

## 数据结构

### 新增接口
```typescript
// GET /api/reminders - 获取提醒设置
interface GetRemindersResponse {
  code: number;
  data: {
    enabled: boolean;
    times: string[];      // ['09:00', '21:00']
    days: number[];       // [0,1,2,3,4,5,6] 0=周一

  };
}

// PUT /api/reminders - 更新提醒设置
interface UpdateRemindersRequest {
  enabled?: boolean;
  times?: string[];
  days?: number[];
  content?: string;
}
```

## 边界情况
1. **通知权限被拒绝**：提示用户去设置开启
2. **App 被卸载重装**：重新订阅通知
3. **手机重启**：系统会自动恢复已订阅的本地通知
4. **多设备登录**：各设备独立设置
5. **提醒时间冲突**：允许同一时间设置多个
6. **日期全部取消**：提示"请至少选择一天"
