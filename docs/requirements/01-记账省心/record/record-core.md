## 功能概述
核心记账功能提供简洁高效的记账体验：
- **支出/收入切换**：通过tab快速切换记账类型
- **分类选择**：网格布局展示分类，分组显示，支持自定义图标
- **金额输入**：大数字键盘，支持简单加减运算
- **日期选择**：支持选择记账日期，默认今天
- **备注功能**：可添加备注信息
- **快速提交**：记账成功后自动跳转明细页

## 用户故事
作为用户，我希望能快速记录一笔支出或收入，选择分类，输入金额，添加备注，选择日期，然后完成记账。

## 交互设计

### 记账页面结构
```
┌────────────────────────────────────┐
│  ← 返回     记一笔                  │
│  ┌──────┬──────┐                 │
│  │ 支出 │ 收入 │                 │ ← 支出/收入tab切换
│  └──────┴──────┘                 │
├────────────────────────────────────┤
│                                     │
│  餐饮组                             │
│  🍔 餐饮  🥤 零食  🍜 面条        │ ← 分类网格布局
│                                     │
│  交通组                             │
│  🚇 地铁  🚕 出租  🚌 公交        │
│  ...                                │
└────────────────────────────────────┘
```

### 金额输入弹窗
点击分类后弹出：
```
┌────────────────────────────────────┐
│          ¥ 0.00                   │ ← 金额显示
│                                     │
│  备注：点击填写                     │ ← 备注输入
│                                     │
│  [7] [8] [9] [ 5-9 ]              │
│  [4] [5] [6] [  +  ]              │ ← 数字键盘
│  [1] [2] [3] [  -  ]              │
│  [.] [0] [⌫] [完成]               │
└────────────────────────────────────┘
```

### 交互流程
1. 用户进入记账页，默认显示支出tab
2. 选择分类（可切换支出/收入）
3. 弹出金额输入表单
4. 输入金额（支持简单加减：如20+30）
5. 可选：点击填写备注
6. 可选：点击日期按钮选择日期
7. 点击「完成」提交
8. 显示「记账成功」提示
9. 1秒后自动跳转明细页

### 状态变化
| 状态 | 触发条件 | 行为 |
|------|----------|------|
| 初始 | 页面加载 | 显示支出tab，清空表单 |
| 切换tab | 点击支出/收入 | 清空已选分类，重新加载分类 |
| 选择分类 | 点击分类 | 弹出金额输入表单 |
| 输入中 | 点击数字/运算符 | 更新金额显示 |
| 提交中 | 点击完成 | 显示loading，防止重复提交 |
| 成功 | 接口返回成功 | 显示成功提示，跳转明细页 |

## UI设计规范

### 页面布局
- **Header**：顶部渐变蓝背景，固定定位
- **内容区**：可滚动，分类网格布局
- **表单弹窗**：从底部滑出，半透明遮罩

### 颜色规范
- **主色调**：#00BFFF（蓝）
- **渐变色**：linear-gradient(135deg, #00BFFF 0%, #0099CC 100%)
- **背景色**：#FAF9F6 → #F5F3EF（米黄色渐变）
- **文字色**：#333（主文字），#999（次要文字）
- **选中状态**：白色图标 + 蓝色渐变背景

### 动效
- **Tab切换**：底部指示器滑入动画
- **分类点击**：scale缩小效果
- **表单弹出**：从底部滑入
- **成功提示**：toast弹窗

## 边界情况

1. **金额为空**：提示「请输入金额」
2. **未选分类**：提示「请选择分类」（表单不会弹出，因为必须先选分类）
3. **重复提交**：提交中禁用按钮，防止重复提交
4. **网络错误**：提示「网络错误」
5. **日期选择**：默认今天，可选择历史日期
