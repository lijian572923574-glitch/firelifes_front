## 功能概述
将常用的记账组合保存为模板（分类+金额+备注+账户），支持一键从模板记账。模板管理页面支持增删改排序。入口：记账页右上角"模板"按钮。

## 用户故事
作为用户，我每周五晚上都会去同一家餐厅吃饭，消费大约200元左右。我希望把这个组合保存为模板，下次记账时一键选择，省去每次重复输入的麻烦。

### 交互流程
1. 用户在记账页点击右上角"模板"按钮
2. 弹出模板列表页面
3. 用户选择模板，点击使用
4. 自动填充模板数据到记账表单
5. 用户可修改后提交
6. 或点击"新建模板"创建新模板
7. 模板支持编辑/删除/排序

### 状态变化
| 状态 | 触发条件 | 行为 |
|------|----------|------|
| 列表页 | 进入模板页 | 显示所有模板 |
| 空状态 | 无模板 | 显示引导创建 |
| 编辑中 | 点击编辑 | 进入编辑表单 |
| 删除确认 | 点击删除 | 显示确认弹窗 |
| 使用成功 | 选择模板 | 填充并关闭页面 |

## 边界情况
1. **分类被删除**：模板显示"分类已删除"提示
2. **账户被删除**：模板保留账户为空
3. **模板名称重复**：允许
4. **模板数量限制**：最多50个模板
5. **使用空金额模板**：用户需手动输入金额
6. **排序更新失败**：回滚本地排序
