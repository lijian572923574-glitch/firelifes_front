## 功能概述
在记账页 Header 区域集成账户选择器，支持从已创建的账户列表中选择本次记账的支付账户。账户数据来自 F1 account-system 账户体系，是记账联动账户余额（F4）的关键入口。

**更新说明**：此功能已从"记账标签"升级为账户体系的一等公民，支持完整的账户类型（资产类/负债类）、实时余额显示、以及与账户管理的联动。

## 用户故事
作为用户，我希望在记账时能指定这笔支出从哪个账户支付，并且能看到该账户的实时余额，这样我可以更清晰地了解各账户的余额变化，确保账实一致。

### 交互流程

1. **打开选择器**
   - 点击 Header 账户选择区域
   - 弹出账户列表弹窗
   - 默认选中上次使用的账户

2. **选择账户**
   - 点击账户项
   - 弹窗关闭
   - Header 更新显示账户图标、名称、余额

3. **搜索账户**
   - 输入账户名称关键词
   - 实时过滤列表
   - 高亮匹配文字

4. **添加新账户**
   - 点击"+ 添加新账户"
   - 进入账户编辑页（F1）
   - 添加成功后返回并选中

5. **管理账户**
   - 点击"管理账户"
   - 进入账户管理页（F1）
   - 可编辑/删除/排序账户

6. **首次使用引导**
   - 检测到无账户
   - 显示引导卡片
   - 提供"一键添加预设账户"

### 状态变化

| 状态 | 触发条件 | 行为 |
|------|----------|------|
| 默认 | 无默认账户 | 显示"选择账户"占位 |
| 已选 | 选择账户后 | 显示账户图标+名称+余额 |
| 无账户 | 用户未创建账户 | 显示空状态引导 |
| 搜索中 | 输入搜索关键词 | 过滤显示匹配账户 |
| 负债 | 信用卡等负余额 | 余额显示红色 |

## 边界情况

1. **无账户**
   - 显示空状态引导
   - 提供"一键添加预设账户"按钮
   - 点击后批量创建5个预设账户

2. **账户余额为负**
   - 信用卡等显示负余额
   - 颜色使用 #FA3534（红色）

3. **账户被删除**
   - 关联记录保留账户快照
   - 不显示在选择列表中

4. **网络错误**
   - 显示本地缓存的账户列表
   - 提示用户刷新

5. **账户名称过长**
   - 截断显示，最多10个字符
   - 显示省略号

6. **上次选择的账户已删除**
   - 回退到第一个账户
   - 或显示"选择账户"

7. **记账时必须选择账户**
   - 未选择时"记账"按钮禁用
   - 提示"请先选择账户"

8. **搜索无结果**
   - 显示"未找到匹配的账户"
   - 提供"添加账户"入口
