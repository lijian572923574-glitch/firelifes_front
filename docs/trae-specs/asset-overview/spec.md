# 资产总览页 - Trae Spec

## Overview
- **Summary**: 资产总览页适配4类资产展示，包括现金类、投资类、固定资产、折旧资产，以及负债类。净资产计算包含4类资产，资产配置环形图展示5个分类，新增固定资产区块和折旧资产区块。
- **Purpose**: 帮助用户在一个页面全面了解个人财务状况，包括各类资产的占比、价值和变化趋势。
- **Target Users**: 所有记账用户

## Goals
- 净资产计算包含4类资产：现金类 + 投资类市值 + 固定资产净权益 + 折旧资产当前价值
- 资产配置环形图展示5个分类：存款（现金类）、投资、房产（固定资产净权益）、折旧资产、负债
- 新增"固定资产"区块：展示固定资产总净权益 + "查看全部"入口
- 新增"折旧资产"区块：展示折旧资产总当前价值 + 本月折旧额 + "查看全部"入口
- 每个账户卡片根据类型显示不同信息

## Non-Goals (Out of Scope)
- 不实现账户的创建/编辑功能（account-system）
- 不实现固定资产的详细管理（fixed-asset）
- 不实现折旧资产的详细管理（depreciating-asset）

## Background & Context
原资产总览只支持2类账户（资产/负债），扩展后需要支持4类资产账户的分类展示和计算。

### 资产配置分类说明

| 分类 | 数据来源 | 金额含义 | 环形图颜色 |
|------|---------|---------|-----------|
| 存款 | 现金类账户余额总和 | 账户余额 | #00BFFF（卡布里蓝） |
| 投资 | 投资类账户市值总和 | 当前市值 | #19BE6B（绿色） |
| 房产 | 固定资产净权益总和 | 估值-关联负债 | #0099CC（深蓝） |
| 折旧资产 | 折旧资产当前价值总和 | 自动折旧计算 | #67C8FF（浅蓝） |
| 负债 | 负债账户余额总和 | 剩余本金 | #FA3534（红色） |

## Functional Requirements

### FR-1: 净资产大字展示
- 顶部大字显示净资产
- 净资产 = 现金类总额 + 投资类市值 + 固定资产净权益 + 折旧资产当前价值 - 负债总额
- 资产总计和负债总计小字展示
- 净资产为负时显示红色

### FR-2: 净资产趋势折线图
- 显示最近12个月的数据点
- X轴：月份，Y轴：净资产金额
- 点击可放大查看

### FR-3: 资产配置环形图
- 展示5个分类的占比
- 点击分类区块显示详细数据
- 各分类颜色区分

### FR-4: 账户卡片列表
- 按类型分组展示
- 资产类：现金类 → 投资类 → 固定资产 → 折旧资产
- 负债类：信用卡/消费信贷 → 房贷/车贷 → 借款
- 点击卡片跳转详情页

### FR-5: 现金类账户卡片
- 显示：账户图标 + 名称 + 余额
- 余额颜色：绿色（资产）

### FR-6: 投资类账户卡片
- 显示：账户图标 + 名称 + 市值 + 收益率
- 收益率颜色：绿色（盈利）/ 红色（亏损）

### FR-7: 固定资产区块
- 展示总净权益
- 每个固定资产卡片显示：资产图标 + 名称 + 净权益 + 增值率
- 超过6个月未更新显示"⚠️估值待更新"
- "查看全部"入口

### FR-8: 折旧资产区块
- 展示当前总价值和本月折旧额
- 每个折旧资产卡片显示：资产图标 + 名称 + 当前价值 + 折旧进度
- "查看全部"入口

### FR-9: 负债类账户卡片
- 显示：账户图标 + 名称 + 余额
- 余额颜色：红色（负债）

## Non-Functional Requirements
- **NFR-1**: 页面加载不超过 300ms
- **NFR-2**: 数字变化使用滚动动画，800ms
- **NFR-3**: 图表加载使用 drawIn 动画，500ms
- **NFR-4**: 支持 UniApp 兼容

## Constraints
- **Technical**: Vue3 + TypeScript + SCSS + Wot UI + UniApp
- **API**: RESTful 风格
- **Design**: 极简、纯白背景、SVG 线框图标、卡布里蓝体系

## Assumptions
- account-system 支持4类资产账户
- depreciating-asset 提供折旧资产汇总数据
- fixed-asset 提供固定资产汇总数据

## Acceptance Criteria

### AC-1: 净资产计算正确
- **Given**: 现金类¥20,000 + 投资类¥50,000 + 固定资产净权益¥1,200,000 + 折旧资产¥5,000 - 负债¥500,000
- **When**: 系统计算净资产
- **Then**: 净资产 = ¥775,000
- **Verification**: `unit-test`

### AC-2: 资产配置环形图展示正确
- **Given**: 存在各类资产数据
- **When**: 查看环形图
- **Then**: 
  - 显示5个分类：存款、投资、房产、折旧资产、负债
  - 各分类占比正确
  - 颜色符合设计
- **Verification**: `human-judgment`

### AC-3: 固定资产区块展示
- **Given**: 存在固定资产数据
- **When**: 查看固定资产区块
- **Then**: 
  - 显示总净权益
  - 每个资产显示：图标 + 名称 + 净权益 + 增值率
  - 超期显示"⚠️估值待更新"
  - 有"查看全部"入口
- **Verification**: `human-judgment`

### AC-4: 折旧资产区块展示
- **Given**: 存在折旧资产数据
- **When**: 查看折旧资产区块
- **Then**: 
  - 显示当前总价值和本月折旧额
  - 每个资产显示：图标 + 名称 + 当前价值 + 折旧进度
  - 有"查看全部"入口
- **Verification**: `human-judgment`

### AC-5: 投资类卡片显示收益率
- **Given**: 某投资账户市值¥52,300，持仓成本¥50,000
- **When**: 查看账户卡片
- **Then**: 显示收益率 +4.6%（绿色）
- **Verification**: `human-judgment`

### AC-6: 净资产为负时红色显示
- **Given**: 净资产 = -¥50,000
- **When**: 查看资产总览
- **Then**: 净资产数字显示红色 #FA3534
- **Verification**: `human-judgment`

### AC-7: 折旧进度显示
- **Given**: 某折旧资产购入价¥5,000，当前价值¥2,500，残值¥500
- **When**: 查看折旧资产卡片
- **Then**: 折旧进度 = (5000-2500)/(5000-500) = 55.6%
- **Verification**: `unit-test`
