## 产品定位

FIRE生活家 是一款面向追求财务自由的年轻用户（尤其是程序员群体）的智能记账应用。通过极简的记账体验、清晰的资产可视化和数据驱动的FIRE进度追踪，帮助用户养成记账习惯、实现财务目标。

## 产品slogan

记账省心、资产有数、FIRE可期

## 架构总览

```
FIRE生活家
├── 记账省心 ────→ 记录每笔收支，关联账户
├── 资产有数 ←─── 记账数据自动更新资产
│   ├── 现金类（余额即价值）
│   ├── 投资类（市值波动）
│   ├── 固定资产（估值+负债）
│   └── 折旧资产（自动折旧+残值）
├── FIRE可期 ←─── 资产+支出数据驱动
│   ├── 目标设定（4%法则）
│   ├── 进度追踪（净资产/目标比）
│   └── 储蓄率&报告
└── 系统配置
    ├── 登录注册
    ├── 设置
    └── 数据同步
```

## 数据流（核心飞轮）

```
记账省心 ──(支出/收入数据)──→ 资产有数 ──(净资产/储蓄率)──→ FIRE可期
    ↑                              |                            |
    └────────(FIRE目标反馈消费决策)←─┘                            |
    └──────────────(月度报告推送)←────────────────────────────────┘
```

## 技术架构

- **前端**：Vue3 + TypeScript + SCSS + Wot UI + UniApp
- **后端**：待定
- **数据存储**：服务端 + localStorage 缓存
- **部署**：微信小程序 / H5

## 模块依赖关系

```
记账省心（基础层，无外部依赖）
  └──→ 资产有数（依赖记账数据）
        └──→ FIRE可期（依赖资产+支出数据）
系统配置（独立模块，无业务依赖）
```
