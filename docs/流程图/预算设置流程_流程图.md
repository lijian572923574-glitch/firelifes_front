## 步骤说明

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 进入发现页 | 从底部导航进入发现页面 |
| 2 | 进入预算管理 | 点击预算管理入口 |
| 3 | 判断已有预算 | 已有预算则查看/调整当前预算；无预算则新建 |
| 4 | 设置预算周期 | 选择每周 / 每月 / 每年作为预算周期 |
| 5 | 设置预算金额 | 输入总预算金额 |
| 6 | 设置分类预算 | 按分类（餐饮、交通、购物等）分别设置预算上限 |
| 7 | 设置超支提醒 | 设置预警阈值（80%/90%/100%），到达阈值时触发提醒 |
| 8 | 保存预算 | 确认保存，写入数据库 |
| 9 | 完成 | 预算设置成功，返回上一页 |

## 预算预警说明

| 阈值 | 状态 | 说明 |
|------|------|------|
| < 60% | 健康  | 支出在预算范围内 |
| 60% ~ 80% | 正常  | 支出正常推进中 |
| 80% ~ 警告阈值 | 警告  | 即将超出预算 |
| ≥ 警告阈值 | 危险  | 已超预算 |
